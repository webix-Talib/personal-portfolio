# AI Systems Architect Portfolio

Premium, production-ready portfolio for an AI Automation Developer and Networking Enthusiast. Built with pure HTML5, Tailwind CSS (CDN), and vanilla JavaScript for fast GitHub Pages deployment.

## Deploy on GitHub Pages

1. Create a new GitHub repository and push this folder.
2. In GitHub, go to **Settings → Pages**.
3. Set **Source** to `main` (or `master`) and root `/`.
4. Save. Your site will be live at `https://<username>.github.io/<repo>/`.

## Custom Domain Setup

1. Buy a domain from your preferred registrar.
2. Add a `CNAME` file in the repository root with your domain (example: `portfolio.your-domain.com`).
3. In your DNS provider, add a CNAME record pointing to `your-username.github.io`.
4. Back in **Settings → Pages**, add your custom domain and enable HTTPS.

## How to Edit Content

- Update text, sections, and links in `index.html`.
- Adjust colors, spacing, and animations in `styles.css`.
- Tweak interactions (nav highlight, scroll reveal, network animation) in `script.js`.
- Replace placeholder URLs (LinkedIn, X/Twitter, and hero metrics) with your real links.

## Asset Notes

Place custom images and icons inside the `assets/` folder and update references in `index.html`.

## SEO Checklist

- Update meta tags (title, description, Open Graph) in `index.html`.
- Replace the `og:image` URL with a real image in `assets/`.
- Set the canonical URL to your real domain.
